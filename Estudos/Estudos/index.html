    Recursos de Aprendizado para Tecnologias Variadas

    Experiência e Interfaces de Usuário (UI/UX)

    1. 10 Heurísticas de Usabilidade para UI - Artigo explicando 10 heurísticas de usabilidade essenciais para design de interfaces de usuário.
    2. UI Patterns - Uma coleção de padrões de interface de usuário para inspirar e informar seu design.
    3. Mapping User Stories in Agile - Artigo explicando sobre o que é e qual a importância do User Stories Mapping.

    Figma e Prototipagem

    1. Curso de Figma Grátis - Tutorial Completo - Curso explicativo de figma gratuito
    2. Curso de Figma - Tutorial - Outro curso sobre figma que aprofunda um pouco mais

    Front-End

    1. Curso para Iniciantes em JavaScript - Série de vídeos para iniciantes cobrindo os fundamentos do JavaScript.
    2. JavaScript - Conceitos Avançados - Vídeos explorando conceitos avançados do JavaScript para aprimorar suas habilidades.
    3. Crie um Site Simples com HTML, CSS e JavaScript - Tutorial passo a passo para criar um site simples usando HTML, CSS e JavaScript.
    4. Curso de HTML e CSS - Vídeo no YouTube - Curso em vídeo para aprender HTML e CSS do zero.

    React

    1. Componentes de UI no React - Recursos e diretrizes para criação de componentes de interface de usuário no React.
    2. Treinamento da Microsoft sobre React - Cursos e materiais de treinamento para aprender React, fornecidos pela Microsoft.
    3. Introdução ao React - Vídeo no YouTube - Vídeo introdutório explicando os conceitos básicos do React.
    4. REACT.JS E AXIOS - CONSUMIR DADOS DE API COM REACT E AXIOS - Curso em vídeo sobre consumir APIs no React.
    5. React Router: O guia completo para navegação em aplicativos React - Curso em vídeo sobre a biblioteca React Router.
    6. Curso de React - Curso em vídeo completo sobre react
    7. Curso de React Para Iniciantes - Curso em vídeo de react para iniciantes

    Back-end

    1. Documentação Django - Documentação do framework que utilizaremos no backend em Python.
    2. Txt2Json - Vídeo explicativo ensinando a fazer a conversão TXT para Json em Python.

    Web Scraping e Raspagem de dados em Python

    1. Introdução Prática ao Web Scraping com Python - Tutorial prático para começar com web scraping usando Python.
    2. Como Fazer Web Scraping em Python - Blog GeekHunter - Guia passo a passo sobre como realizar web scraping em Python.
    3. Web Scraping com Python - Blog BrightData - Recursos e tutoriais para ajudar você a entender e praticar web scraping com Python.
    4. Playlist de Web Scraping em Python - Vídeos no YouTube - Série de vídeos abrangendo diferentes aspectos do web scraping usando Python.
    5. Scrapy - Vídeo-aula introdutória ao Scrapy
    6. BeatifulSoup - Vídeo-aula introdutória ao BeatifulSoup
    7. Selenium - Vídeo-aula introdutória ao Selenium

    Biblioteca PyPDF2 (Python)

    1. Ler texto, imagem, colocar senha e dar merge em PDF - Playlist de tutorial (inglês).
    2. Como Ler Arquivo PDF, Como Mesclar (merge) arquivos PDF, PDF para Texto com Python , PyPDF2 - Vídeo de tutorial (português).
    3. Obtendo metadados, texto, colocando e removendo senha - Playlist mais completa (metadados, senha, merge e texto).
